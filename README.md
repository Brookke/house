# House Cost Calculator

A UK house buying cost calculator that helps you understand the true cost of purchasing a property in England and Wales.

**Live site:** [house.brookehatton.com](https://house.brookehatton.com)

## Features

- **Stamp Duty Calculator** - Calculates stamp duty land tax with support for first-time buyer relief
- **Legal Fees Estimate** - Realistic UK conveyancing fee estimates including VAT and disbursements
- **Monthly Payment Breakdown** - Visualizes mortgage payments, service charges, and ground rent
- **Shareable Links** - All calculator inputs sync to the URL, making results easy to share

## Tech Stack

- [React](https://react.dev/) 19
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [TanStack Router](https://tanstack.com/router)
- [Mantine](https://mantine.dev/) UI
- [Tabler Icons](https://tabler.io/icons)

## Development

```bash
# Install dependencies
npm install

# Start dev server on port 3000
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run tests
npm run test
```
